# Carpeta `node_modules/`

Dependencias instaladas automáticamente por `npm install`, a partir de lo
declarado en `package.json` y fijado en `package-lock.json`. Sólo hay dos
paquetes reales detrás de todo esto (`electron` y `electron-builder`); el resto
son sus propias dependencias internas.

**No se edita nada a mano aquí.** Si algo se ve raro (errores extraños al
ejecutar `npm start`), lo primero es borrar la carpeta y volver a instalar:

```powershell
Remove-Item -Recurse -Force node_modules
npm install
```

Esta carpeta no se sube a control de versiones (si en algún momento se agrega
Git al proyecto, debe ir en `.gitignore`).
